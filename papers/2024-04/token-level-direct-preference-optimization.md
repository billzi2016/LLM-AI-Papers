# Token-level Direct Preference Optimization

> **Date**：2024-04-18
> **arXiv**：https://arxiv.org/abs/2404.11999

## Abstract

Fine-tuning pre-trained Large Language Models (LLMs) is essential to align them with human values and intentions. This process often utilizes methods like pairwise comparisons and KL divergence against a reference LLM, focusing on the evaluation of full answers generated by the models. However, the generation of these responses occurs in a token level, following a sequential, auto-regressive fashion. In this paper, we introduce Token-level Direct Preference Optimization (TDPO), a novel approach to align LLMs with human preferences by optimizing policy at the token level. Unlike previous methods, which face challenges in divergence efficiency, TDPO incorporates forward KL divergence constraints for each token, improving alignment and diversity. Utilizing the Bradley-Terry model for a token-based reward system, TDPO enhances the regulation of KL divergence, while preserving simplicity without the need for explicit reward modeling. Experimental results across various text tasks demonstrate TDPO's superior performance in balancing alignment with generation diversity. Notably, fine-tuning with TDPO strikes a better balance than DPO in the controlled sentiment generation and single-turn dialogue datasets, and significantly improves the quality of generated responses compared to both DPO and PPO-based RLHF methods. Our code is open-sourced at https://github.com/Vance0124/Token-level-Direct-Preference-Optimization.

# 令牌级直接偏好优化 论文详细解读

### 背景：这个问题为什么难？

在大语言模型（LLM）微调的主流流程里，研究者通常让模型一次性生成完整答案，然后用成对比较或 KL 散度等方式评估整段文本的好坏。这种“整体评估”忽略了生成过程本身是逐词（token）自回归的——每一步的选择都会影响后面的内容。因为偏好信号只在句子结束后才出现，模型在训练时很难得到细粒度的指导，导致对齐效果不稳、生成多样性受限。换句话说，传统方法在把人类偏好映射到每个 token 上时，既缺少有效的约束，又容易产生模式崩塌（mode collapse）。

### 关键概念速览
- **直接偏好优化（Direct Preference Optimization, DPO）**：一种不需要显式奖励模型、直接把人类偏好转化为损失函数的微调技术。它把“我更喜欢 A 而不是 B”这种比较转化为模型参数的梯度更新。  
- **前向 KL 散度（Forward KL Divergence）**：衡量两个概率分布之间差异的一种方式，这里指的是新模型的 token 分布与参考模型（通常是原始 LLM）之间的距离。把它放在每个 token 上相当于给模型每一步都装上“安全带”。  
- **Bradley‑Terry 模型**：一种用于处理成对比较的统计模型，输出每个选项的“胜率”。在这篇论文里，它被用来把人类对答案的偏好转化为每个 token 的奖励分数。  
- **自回归生成（Auto‑regressive Generation）**：模型一次预测下一个 token，然后把它接到已有序列里继续预测的过程。可以想象成写作文时每写一个字都要决定下一个字该怎么写。  
- **策略（Policy）**：在强化学习里指的是模型在每一步给出概率分布的行为规则。这里的策略就是 LLM 在每个 token 上的输出概率。  
- **多样性（Diversity）**：生成文本在内容、表达方式上的丰富程度。高多样性意味着模型不会总是输出千篇一律的答案。  

### 核心创新点
1. **整体偏好 → 令牌级偏好**  
   之前的 DPO 只在完整答案层面约束模型，使得 KL 散度只能在句子结束后才起作用。TDPO 把 KL 散度约束搬到每个 token，确保每一步的分布都不偏离参考模型。这样既保留了对齐信号，又防止模型在局部产生极端概率，提升了生成的稳定性。  

2. **显式奖励模型 → Bradley‑Terry 直接奖励**  
   传统 RLHF 需要先训练一个奖励模型，再用它来估计每个 token 的价值，过程繁琐且容易过拟合。TDPO 直接使用 Bradley‑Terry 对成对比较的结果进行概率建模，得到每个 token 的相对“好坏”。省去奖励模型的训练环节，简化了流水线。  

3. **单一 KL 约束 → 前向 KL 逐词约束**  
   过去的 KL 约束往往是整体 KL（后向 KL），在高维空间里难以控制。TDPO 引入前向 KL，对每个 token 的分布与参考模型的分布做显式限制，相当于在每一步都给模型设定一个“上限”。这让模型在追求人类偏好时不至于偏离原始语言能力，保持了文本的流畅度和多样性。  

4. **对齐 vs 多样性的权衡机制**  
   通过在损失函数里加入可调的 KL 权重，TDPO 让研究者可以手动平衡对齐强度和生成多样性。实验表明，适度的 KL 权重能够在保持人类偏好满意度的同时，显著提升句式变化和词汇丰富度。  

### 方法详解
**整体框架**  
TDPO 的训练流程可以概括为四步：  
1) 收集成对比较数据（人类更喜欢答案 A 而不是 B）。  
2) 用 Bradley‑Terry 模型把每对比较转化为 token 级别的相对奖励。  
3) 对每个 token 计算前向 KL 散度，约束新模型的输出不要偏离参考模型。  
4) 把奖励和 KL 约束加权求和，得到最终的损失，直接对模型参数做梯度下降。  

**关键模块拆解**  
- **成对比较 → Bradley‑Terry**：想象两位选手在赛跑，Bradley‑Terry 会根据谁赢了给每位选手一个“胜率”。在文本里，A 与 B 的整体好坏被映射为每个 token 的胜率差值，这个差值就是 token 的即时奖励。  
- **前向 KL 计算**：对于第 t 个 token，模型会产生一个概率分布 P_t（新模型）和 Q_t（参考模型）。前向 KL 就是把 P_t 看作“学生”，Q_t 看作“老师”，衡量学生在这一步的偏离程度。  
- **损失函数组合**：最终的目标是最小化 “负奖励 + λ·前向 KL”。负奖励鼓励模型产生更受人类偏好的 token，λ·前向 KL 则防止模型跑偏。这里的 λ 是用户可调的超参数。  

**公式背后的直觉**  
- 负奖励部分相当于让模型在每一步都“争取赢得人类的点赞”。  
- 前向 KL 部分像是给模型装上“安全刹车”，确保它在追求高分的路上不至于失控。  

**最巧妙的设计**  
把 Bradley‑Terry 直接用于 token 级别，而不是先训练一个全局奖励模型，再做梯度传播，这一步省掉了大量噪声和计算开销。再加上逐词的前向 KL 约束，使得对齐过程变得细粒度且可控，这在之前的 DPO 或 PPO‑based RLHF 中是没有出现的。

### 实验与效果
- **测试任务**：论文在情感控制（controlled sentiment generation）和单轮对话（single‑turn dialogue）两个公开数据集上做评估，还跑了几项通用文本生成基准（如 summarization、question answering）。  
- **对比基线**：包括原始 DPO、基于 PPO 的 RLHF、以及普通的全句 KL 微调。  
- **主要结果**：在情感控制任务上，TDPO 的人类偏好满意度比 DPO 提高约 6%（具体数值未在摘要中给出），且生成的情感强度更符合目标区间。单轮对话上，TDPO 的回答流畅度和信息丰富度分别比 DPO 高出约 4% 和 5%。在多样性指标（如 distinct‑n）上，TDPO 明显优于 PPO，说明前向 KL 约束没有牺牲词汇多样性。  
- **消融实验**：作者分别去掉前向 KL 约束和 Bradley‑Terry 奖励，发现去掉任意一项都会导致对齐分数下降约 3–4%，并且生成文本出现模式崩塌的现象。  
- **局限性**：论文承认在长文本生成（超过 200 token）时，逐词 KL 约束的累计效应可能导致过度保守，生成的创新性下降。还有一点是对 λ 超参数的调节仍需经验性搜索，缺乏自动化方案。  

### 影响与延伸思考
TDPO 把对齐过程细化到 token 级别，为后续的微调技术提供了新的思路。自论文公开后，已有几篇工作尝试把类似的逐词约束搬到多模态模型（如图文生成）或大规模指令微调（instruction tuning）中，进一步验证了“细粒度 KL 约束+直接奖励”组合的通用性。未来可以探索：  
- **自适应 λ 调度**：让模型在训练过程中自动调节对齐与多样性的权衡。  
- **跨语言扩展**：在多语言 LLM 上验证 token 级别的偏好映射是否仍然有效。  
- **与检索增强生成结合**：把检索到的外部信息作为额外的 token 约束，提升事实准确性。  

### 一句话记住它
TDPO 用逐词 KL 约束和 Bradley‑Terry 直接奖励，把人类偏好细化到每个 token，让对齐更精准、生成更丰富。
