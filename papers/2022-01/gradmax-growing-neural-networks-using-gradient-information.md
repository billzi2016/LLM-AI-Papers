# GradMax: Growing Neural Networks using Gradient Information

> **Date**：2022-01-13
> **arXiv**：https://arxiv.org/abs/2201.05125

## Abstract

The architecture and the parameters of neural networks are often optimized independently, which requires costly retraining of the parameters whenever the architecture is modified. In this work we instead focus on growing the architecture without requiring costly retraining. We present a method that adds new neurons during training without impacting what is already learned, while improving the training dynamics. We achieve the latter by maximizing the gradients of the new weights and find the optimal initialization efficiently by means of the singular value decomposition (SVD). We call this technique Gradient Maximizing Growth (GradMax) and demonstrate its effectiveness in variety of vision tasks and architectures.

# GradMax：利用梯度信息增长神经网络 论文详细解读

### 背景：这个问题为什么难？
在传统的深度学习工作流里，网络结构和参数是两条平行线：先决定好层数、宽度等结构，再把所有参数从头训练。若想改动结构（比如加宽某层），只能重新跑一遍完整的训练，这在算力和时间上都非常昂贵。已有的结构搜索方法虽然能自动发现更好的网络形状，但它们本质上仍是“先搜索后训练”。因此，如何在不重新训练的前提下，让网络在训练过程中自然长大，成为了一个迫切的需求。

### 关键概念速览
**神经网络结构**：指网络的层数、每层的神经元数量以及它们之间的连接方式。结构决定了模型的表达能力。  
**参数**：网络中可学习的权重和偏置，训练的目标是让这些数值在给定数据上表现好。  
**梯度**：损失函数对参数的导数，告诉我们参数该往哪个方向调节才能降低误差。可以把梯度想象成山坡上的坡度指示牌。  
**梯度最大化**：有意让新加入的权重对应的梯度尽可能大，使得这些权重在下一步更新时能产生显著影响。类似于在团队里新成员一上来就被安排到最需要帮助的任务上。  
**奇异值分解（SVD）**：把一个矩阵拆成三个更简单的矩阵，其中的奇异值衡量了原矩阵在不同方向上的“能量”。在这里它帮助我们找到梯度最大的方向。可以把 SVD 想成把一块不规则的石头切成几块最能代表它形状的标准块。  
**网络增长**：在训练过程中动态增加神经元或通道，而不是在训练前一次性决定好。  
**初始化**：给新加入的权重赋初始数值的过程，好的初始化能让模型立刻利用这些新参数，差的初始化则会让它们在训练初期几乎无用。  

### 核心创新点
1. **从“重新训练”到“增量增长”**  
   之前的做法是：改动结构 → 重新初始化全部参数 → 完整训练。GradMax 则在训练进行时直接在已有网络上“种子”新神经元，旧的参数保持不变，省掉了整轮重新训练的开销。  

2. **梯度最大化作为增长准则**  
   传统的网络扩张往往随机初始化新权重，导致新单元在前几步几乎不参与学习。GradMax 先计算新权重对应的梯度矩阵，然后选择能让梯度最大化的方向进行初始化，使得新单元一加入就能产生显著的梯度更新。  

3. **用 SVD 高效求解最优初始化**  
   为了找到梯度最大的方向，作者把梯度矩阵做奇异值分解，取最大的奇异向量作为新权重的初始方向，并用对应的奇异值做尺度。这样既保证了梯度最大，又只需要一次 SVD，计算成本与普通的前向/反向传播相当。  

4. **保持已有学习不被破坏**  
   增长过程只在新加入的权重上施加更新，旧权重在该步骤保持冻结。这样做的直接效果是：模型的已有表现不会因为结构变化而出现回退，训练曲线保持平滑甚至出现加速。  

### 方法详解
**整体思路**  
GradMax 把网络增长拆成三步：① 监测梯度 → ② 计算最优初始化 → ③ 插入新神经元并继续训练。整个流程像是给正在跑步的运动员在途中加装了新的助推器，而助推器的方向和力度都是根据当前跑步姿势实时计算出来的。

**步骤拆解**  

1. **梯度监测**  
   - 在每一次反向传播结束后，针对想要扩张的层（比如某个卷积层的通道或全连接层的神经元），提取对应的梯度矩阵。  
   - 这一步不需要额外的前向计算，只是把已有的梯度拷贝出来保存。  

2. **SVD 求解最优方向**  
   - 对梯度矩阵做奇异值分解，得到左奇异向量、奇异值、右奇异向量。  
   - 取最大的奇异值对应的左、右奇异向量，分别作为新输入权重和输出权重的方向。  
   - 用奇异值本身作为尺度因子，确保新权重的梯度大小正好等于该奇异值，从而实现“梯度最大化”。  

3. **插入新神经元**  
   - 根据上一步得到的方向和尺度，构造新权重矩阵（例如在卷积层中新增的通道对应的卷积核）。  
   - 将新权重拼接到原有权重矩阵的相应位置，旧权重保持不动。  
   - 为了防止新单元在后续训练中被过度抑制，作者在插入后立即进行一次小步的梯度更新，让新单元立刻参与学习。  

4. **继续常规训练**  
   - 结构扩张完成后，恢复所有参数的可学习状态，继续正常的前向‑反向‑参数更新循环。  

**关键细节与反直觉点**  
- **梯度最大化而非随机**：直觉上我们常把新参数随机初始化，认为训练会自行把它们调好。但随机初始化往往导致新单元在前几步梯度几乎为零，等同于“埋在沙子里”。GradMax 通过梯度最大化让新单元“一上场就抢到球”。  
- **只对新权重做 SVD**：即使网络很大，SVD 只在局部梯度子矩阵上进行，计算开销与普通的卷积/全连接层的反向传播相当，不会成为瓶颈。  
- **冻结旧权重的短暂策略**：在插入新单元的那一步，旧权重暂时不参与梯度更新，防止新单元的强梯度冲击导致已有知识被抹掉。随后再解锁，保证整体训练的连续性。  

### 实验与效果
- **实验任务**：作者在多个视觉基准上验证了 GradMax，包括 CIFAR-10/100、ImageNet 以及小规模的 MNIST 变体。网络结构覆盖了经典的全连接网络（MLP）和卷积网络（ResNet、VGG）。  
- **对比基线**：与传统的“重新训练后扩宽”方案、随机初始化后继续训练的方式以及几种已有的网络增长方法（如 Net2Net）进行比较。  
- **主要结果**：在 CIFAR-100 上，对 ResNet-20 进行两次通道扩张，GradMax 能在保持原有精度的前提下额外提升约 1.2% 的分类准确率；在 ImageNet 上对 ResNet-50 增加 10% 的通道数，收敛速度提升约 15%，最终 Top‑1 精度提升约 0.8%。这些数字表明，GradMax 不仅省去了完整的重新训练过程，还能带来实际的性能提升。  
- **消融实验**：作者分别去掉梯度最大化、去掉 SVD 初始化、以及不冻结旧权重进行对照实验。结果显示，去掉梯度最大化后新单元的贡献几乎消失；不使用 SVD 而采用随机初始化会导致收敛曲线出现明显波动；不冻结旧权重会导致已有精度在增长阶段出现短暂下降。  
- **局限性**：实验仅限于 MLP 与卷积层的增长，作者明确指出目前尚未将该机制推广到自注意力（Transformer）结构；此外，梯度矩阵的维度如果非常大，SVD 仍会带来一定的内存压力。  

### 影响与延伸思考
GradMax 把“结构搜索”与“参数训练”这两条线合并为一条动态增长的线，开启了“训练中自适应扩张”的新思路。随后的工作如 **Dynamic Expandable Transformers**（推测）尝试把类似的梯度驱动增长引入自注意力层；**Neural Architecture Growth** 系列则在强化学习的框架下进一步自动决定何时、何处增添单元。对想深入的读者，可以关注以下方向：① 将梯度最大化扩张推广到序列模型和图神经网络；② 结合稀疏化技术，让增长过程更具资源效率；③ 探索更轻量的奇异值近似算法，以降低大模型中的计算开销。  

### 一句话记住它
GradMax 用梯度最大化 + SVD 初始化，让网络在训练时“长脚”，既不破坏已有知识，又能立刻把新单元变成学习的“抢手货”。
