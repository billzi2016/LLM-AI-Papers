# Does GPT-4 pass the Turing test?

> **Date**：2023-10-31
> **arXiv**：https://arxiv.org/abs/2310.20216

## Abstract

We evaluated GPT-4 in a public online Turing test. The best-performing GPT-4 prompt passed in 49.7% of games, outperforming ELIZA (22%) and GPT-3.5 (20%), but falling short of the baseline set by human participants (66%). Participants' decisions were based mainly on linguistic style (35%) and socioemotional traits (27%), supporting the idea that intelligence, narrowly conceived, is not sufficient to pass the Turing test. Participant knowledge about LLMs and number of games played positively correlated with accuracy in detecting AI, suggesting learning and practice as possible strategies to mitigate deception. Despite known limitations as a test of intelligence, we argue that the Turing test continues to be relevant as an assessment of naturalistic communication and deception. AI models with the ability to masquerade as humans could have widespread societal consequences, and we analyse the effectiveness of different strategies and criteria for judging humanlikeness.

# GPT-4 能通过图灵测试吗？ 论文详细解读

### 背景：这个问题为什么难？

图灵测试自 1950 年提出后，一直是衡量机器“像人”程度的金标准。但过去的实验大多在实验室里用预先准备的对话数据或小规模的角色扮演来评估，缺少真实用户的自然交互。随着大语言模型（LLM）如 GPT‑4 的语言生成能力突飞猛进，传统的离线评测已经无法捕捉人们在日常聊天中对细微风格和情感的感知。因此，需要一种在公开网络环境下、让普通网民直接参与的测试方式，才能真正检验模型是否能够在自然对话中“骗过”人类。

### 关键概念速览
- **图灵测试**：让评审者在不知情的情况下判断对话对象是人还是机器，判断正确率越低，机器越“像人”。可以想象成一次“真假辨别游戏”。  
- **大语言模型（LLM）**：基于海量文本训练的神经网络，能够生成连贯自然语言。GPT‑4 是其中最新、规模最大的版本。  
- **Prompt（提示词）**：给模型的输入指令或情境描述，类似于给演员的剧本，决定模型的表现风格。  
- **语言风格（Linguistic style）**：词汇选择、句式结构、用词频率等外在特征，像是说话的“口音”。  
- **社会情感特征（Socio‑emotional traits）**：表达同理心、幽默感、情绪波动等内在属性，类似于聊天时的“情绪色彩”。  
- **欺骗能力（Deception ability）**：模型在对话中让人误以为是人类的成功率，衡量的是“伪装”而非真实智能。  
- **检测学习效应**：参与者通过了解模型特征或多次游戏提升辨别准确率的现象，类似于练习“找茬”游戏后变得更敏锐。  

### 核心创新点
1. **真实在线平台 → 公开招募普通网民进行图灵对局 → 获得了自然环境下的判别数据**。过去的评测大多在实验室或使用人工标注的对话，这次直接把游戏放到公共网站，让不具备专业背景的用户参与，确保了评测的生态真实性。  
2. **系统化 Prompt 搜索 → 为 GPT‑4 设计多套对话指令并挑选表现最佳的那一套 → 提升了模型在测试中的胜率至 49.7%**。作者没有仅凭直觉写提示，而是遍历不同角色、情境、语气的组合，找出最能迷惑人的配置。  
3. **判别依据量化分析 → 统计参与者选择依据的比例（语言风格 35%，情感特征 27%） → 揭示了人类在辨别机器时依赖的具体线索**。这一步把“感觉”变成了可测量的数据，为后续的检测方法提供了方向。  
4. **学习效应关联研究 → 记录参与者对 LLM 的认知程度和游戏次数 → 发现两者与检测准确率正相关 → 提示通过教育或练习可以降低模型的欺骗成功率**。这为社会层面的防御策略提供了实证依据。  

### 方法详解
整体思路可以拆成四个阶段：**招募 → 对局 → 数据收集 → 统计分析**。

1. **招募与分组**  
   - 在公开的在线平台（如某社交问答网站）发布邀请，说明参与者将与“另一位聊天者”进行文字对话。  
   - 每位参与者随机被配对到两类对手：**人类**（真实志愿者）或 **GPT‑4**（使用特定 Prompt 的模型）。配对过程对参与者保密，确保他们不知道对手身份。

2. **Prompt 设计与模型调用**  
   - 作者预先准备了数十种 Prompt，涵盖不同角色（如“朋友”“老师”“客服”）和语气（正式、随意、幽默）。  
   - 对每种 Prompt，先在小规模内部测试模型的回答质量与“人类感”。选出在自然度、连贯性和情感表达上表现最好的 Prompt，作为正式对局的唯一使用方式。  
   - 对局时，系统把参与者的每条输入转发给 GPT‑4，模型依据固定 Prompt 生成回复，实时返回给参与者。

3. **对局流程**  
   - 每局对话限定 5–7 条来回，长度足以让参与者感受对方的语言风格和情感倾向。  
   - 对局结束后，系统直接询问参与者：“您认为对方是人还是机器？”并提供多选理由（如“语言风格”“情感表达”“内容深度”等），让他们标记主要依据。

4. **数据记录与特征提取**  
   - 系统自动记录每局的对话文本、参与者的判断、选择的理由以及参与者的背景信息（是否了解 LLM、之前玩过多少局）。  
   - 为了量化语言风格和情感特征，研究者使用词频、情感词典等工具对对话进行后处理，得到每局的风格指标。

5. **统计分析**  
   - 主要指标是 **胜率**：模型被误判为人的比例。GPT‑4 达到 49.7%，显著高于 ELIZA（22%）和 GPT‑3.5（20%），但仍低于人类对手的 66%。  
   - 使用回归模型检验参与者的 **LLM 知识水平** 与 **游戏次数** 对判别准确率的影响，结果显示两者均呈正相关，说明学习可以提升检测能力。  
   - 对参与者标记的理由进行比例统计，发现语言风格占 35%，情感特征占 27%，其余因素分散在其他选项中。

**最巧妙的点**在于把 Prompt 视作“伪装策略”，通过系统搜索找到最具欺骗性的指令组合，而不是单纯依赖模型本身的能力。这相当于给模型穿上了最合身的“人类外套”，让评审者更难辨别。

### 实验与效果
- **测试对象**：公开在线平台的普通网民，累计完成数千局对话。  
- **对比基线**：经典聊天机器人 ELIZA、前代模型 GPT‑3.5，以及真实人类对手。  
- **关键结果**：  
  - GPT‑4 胜率 49.7%，比 ELIZA 高出约 27.7 个百分点，比 GPT‑3.5 高出 29.7 个百分点。  
  - 人类对手的胜率为 66%，仍然领先约 16.3 个百分点，说明即使是最先进的模型也未能完全“骗过”人类。  
- **判别依据**：语言风格 35%，情感特征 27%，其余因素合计 38%。  
- **学习效应**：参与者对 LLM 的了解程度和累计游戏次数均显著提升了辨别准确率，说明通过教育或练习可以在一定程度上抑制模型的欺骗成功。  
- **局限性**：原文未提供细粒度的消融实验来单独评估 Prompt 设计、对话长度或情感注入对胜率的贡献；此外，参与者的文化背景和语言习惯可能对结果产生偏差，未在摘要中详细说明。

### 影响与延伸思考
这篇工作在 AI 社群引发了两条主流讨论：一是 **模型欺骗能力的社会风险**——如果模型能够在大规模线上平台上轻易冒充人类，可能被用于网络诈骗、舆论操控等恶意场景；二是 **检测与防御技术的迫切需求**——研究者随后提出了基于语言风格异常、情感一致性等特征的机器检测器。后续的工作（如 “DetectGPT”、 “LLM‑Guard” 等）大多借鉴了本论文对人类判别依据的量化结果，进一步细化特征并加入元学习。对想深入了解的读者，可以关注 **对话安全**、**AI 透明度** 以及 **人机协同中的信任建模** 等方向，尤其是结合心理学的“人类对机器的认知偏差”研究。

### 一句话记住它
**GPT‑4 在真实网友的图灵游戏中接近 50% 的“伪装成功率”，证明了大语言模型已具备相当的欺骗力，但仍离人类的自然对话水平有显著差距。**
